icon_pick

用于选择 ICON

使用示例

在 html 中使用

<icon-picker name="icon" value="bi bi-123"></icon-picker>

在 python 中使用

from dwc.widgets import IconPicker

class AdminMenuForm(forms.ModelForm):
    icon = forms.CharField(label="图标", strip=True, widget=IconPicker)
